# Commands

<h3>/autopost</h3>

> Start, Stop, Confirm, or test auto post settings.

<h3>/gta</h3>

> Get the latest GTA Online bonuses

<h3>/help</h3>

> A list of commands

<h3>/language</h3>

> Change the language for the current server.

> The bot responds with "pong" when online
